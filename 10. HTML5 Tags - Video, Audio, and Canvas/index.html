Video

Video is added with the video tag. It will take a source inside of the tag. A full list of video tags and methods can be found at W3 Schools.

Background Opacity

You may want to make your divs transparent so you can display text over the top of your video. This can be done with the opacity tag, but for a better effect, use the rgba() function to set color and opacity of the background without affecting the text.

Audio Tags

The audio element can be used to add sound to the page. Using the controls tag inside of the element will allow the user to control the sound, which we have done here.

Canvas

Canvas has many uses. It uses JavaScript to draw on the webpage, either in 2D or 3D. This has a variety of applications, from advertising to games to custom animation. The best part of all: canvas works accross all modern browsers and platforms, so once you've got it working, it will work for all clients (provided they have a modern, updated browser).
