JSON

JavaScript Object Notation (JSON) is a way to write JavaScript objects in string form. This allows JS Objects to be passed from server to client and transformed back into an object for use in the code. To turn an object into text (ready to send), use JSON.stringify(). To translate a JSON string back into an object, use JSON.parse().
