Object Creation Functions

Objects are like variables, except that they contain other variables. They are like Associative Arrays, with names for each piece of data stored in them. Objects can contain functions as well as variables. An object can be created using curly brackets, or by using a function.

Instantiation

Instantiation is creating an object from an object constructor. This creates a named instance of the object with unique values for each property in the object definition. After creation, the object's properties and methods can be accessed using the object's name.property.

Inheritance

Typical programming languages use classes to define an object, and when defining a new object class, you can say that it inherits from another class. This is called class-based inheritance. JavaScript uses a method called prototype-based to inherit properties, which differs from the class-based inheritance. Every object in JS has a prototype object attached to it that defines what methods are inheritable and what it inherits from.

This greeting function, since it is defined in the myObject prototype will now be inherited by objects that inherit from myObject.

This new InheritedObject should now have all properties from myObject's constructor, and it will inherit all methods defined in the prototype of myObject.

Properties

Properties of an object are the variables stored in the object. Above, in the first example, firstName and lastName are properties of myObject.

Methods

Methods are functions that are stored in an object. Methods are inheritable if they are defined using the .prototype notation on the object i.e. myObject.prototype.functionName. The greeting function above is an example of a method, a function inside of an object.
