Localstorage

Localstorage is a perisistent storage object, stored in the browser, that lasts between sessions. This allows data to be stored and retrieved the next time the user opens the page. Here we will store the color and font as simple data, and then we will store the other values as an array/object (because they are so similar in JS). A few catches: all values must be stored as key/value pairs, and all values must be strings. This is where JSON.stringify will come in handy.
