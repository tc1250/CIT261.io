Associative Arrays

In most programming languages, associative arrays are key:value pairs stored in an array. Example, I can store pet:rufus as a key value pair in a pets array, and when I call pets[pet], I'll get rufus as the return value. However, in JavaScript, when key value pairs are used, the array actually becomes an object, so an object can be created that acts as an associative array, but it is not truly an array in JavaScript, it is an object.
