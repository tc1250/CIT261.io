HTML Events

HTML events are things that happen on an HTML page. JavaScript can react to these events. HTML events can be initiated by the browser or the user.

Common events include: onchange, onclick, onmouseover, onmouseout, onkeydown, onload, and on mobile, touchstart and touchend are important when considering touch events.
